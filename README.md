# Next-Blog BETA-DEV
## 此项目仍在测试，不保证可用性，请候正式版发布后再使用。
### Powered by [Gmeek](https://github.com/Meekdai/Gmeek)

基于Gmeek的博客系统，使用`Github Pages` + `Github Issues` + `Github Actions` 快速搭建个人博客。

### 快速开始

- 点击[Use this template](https://github.com/MurthiNext/NextBlog/generate)按钮，创建自己的仓库。
- 进入仓库的`Settings`页面，找到`Pages`设置，将`Source`设置为`master branch / docs folder`，点击`Save`保存设置。
- 等待几分钟，Github Pages会自动生成静态页面，访问`https://<username>.github.io/NextBlog/`即可看到博客页面。
- 进入仓库的`Issues`页面，点击`New issue`，创建第一篇文章。
